Slack Attendance App

The Slack Attendance Tracker is an app that I have been developing on the side for the last few months with plans to have it deployed by the end of this year. My mission in developing this app is to streamline the process of taking attendance for any organization that uses Slack for communication. I achieve this by using the Slack API to allow users to create events and mark them as attended by reacting to an automated message.

Y86 CPU

Y86 assembler language is a simplified version of X86. I designed a microprocessor from scratch in a circuit drawing software called Logisim. The processor is capable of running basic scripts written in the Y86 assembler language. This project was my first taste of computer organization and low level programming. This project proved to be very difficult but ended up being incredibly rewarding in the end.
